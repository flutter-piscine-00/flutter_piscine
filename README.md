# Dart-Flutter piscine (ver_0.1)

## 목표
- 플러터를 활용하여 아이디어를 빠르게 어플리케이션으로 구현할 수 있다. ⚡️
- 하나의 프로그래밍 언어로 IOS, Android 두 OS의 어플리케이션을 동시에 만들 수 있다. 🧙‍♀️
- 마켓에 등록할 수 있는 수준의 아름다운 어플리케이션을 만들 수 있다. ⭐️
